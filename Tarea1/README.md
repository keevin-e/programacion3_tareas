# Tarea 1

Instrucciones específicas para ejecutar esta tarea.

## Requisitos
- Haber creado el entorno virtual global en la raíz del repo (`.venv`). opcional
- Tener el entorno virtual activo.

## Ejecutar
Desde la raíz del repositorio:

```bash
source .venv/bin/activate
cd Tarea1
python main.py
```

## Funcionamiento

- al ejecutar el programa se iniciara un menu con 5 opciones
    1. insertar el principio: se inserta un nodo al principio de la lista doble y se agrega un nodo al grafico de graphviz automaticamente
    2. insertar al final: se inserta un nodo nuevo al final de la lista doble y se agrega un nodo al grafico de graphviz automaticamente
    3. eliminar un nodo por su carnet: se elimina un nodo por su carnet y se actualiza el grafico.
    4. Mostrar lista: se grafica de una estructura especial la lista doblemente enlazada 
    5. Salir del programa
